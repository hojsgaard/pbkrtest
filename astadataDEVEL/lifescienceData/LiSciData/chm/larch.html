larch(dataRep)R Documentation

Japanese larch data

Description

Height, ash content and content of nitrogene, phosphoreus and calcium for 26 Japanese larch trees.

Usage

data(larch)

Format

A data frame with 26 observations on the following 5 variables.

nit
a numeric vector
phos
a numeric vector
calc
a numeric vector
ash
a numeric vector
height
a numeric vector
