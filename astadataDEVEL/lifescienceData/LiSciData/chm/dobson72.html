dobson72(dataRep)R Documentation

Beetles killed by disulphide

Description

Effect of gaseous carbon disulphide on beetles after 5 hours of exposure.
binary logistic regression

Usage

data(dobson72)

Format

Data frame containing

dose:
dose of disulphide: [log10 of CS2 measured in mg/l]
nbeetle:
number of beetles exposed
nkilled:
number of beetles died

Details

Data are from the exercise Table 7.2 of A. Dobson, page 119.

Source

Dobson, A. (2002) An introduction to Generalized Linear Models, London: Chapman & Hall

Examples

data(dobson72)
summary(dobson72)
