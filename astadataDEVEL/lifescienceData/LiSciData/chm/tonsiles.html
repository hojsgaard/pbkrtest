tonsiles(dataRep)R Documentation

reaction on chemotherapy with tonsiles

Description

Size of tonsiles for corries and non-carries of Streptococcus pygenes

Usage

data(tonsiles)

Format

This data frame contains the following columns:

carrier:
yes: patient carries Streptococcus pygenes; no: patient does not carry the bacterium
size:
size of tonsile: 0: present but not enlarged 1: enlarged 2: greatly enlarged
n
number of persons

Details

Source

McCullagh, P. (1980) Regression models for ordinal data JRSS,B, 109-142

References

Examples

require(dataRep)
data(tonsiles)
