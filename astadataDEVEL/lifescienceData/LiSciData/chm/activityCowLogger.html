activityCowLogger(dataRep)R Documentation

Daily activity of cows.

Description

The cows' id for the logger used for daily activity measurments of the cows. A subset of the data activityCowsLong.

Usage

data(activityCowLogger)

Format

A dataframe with 2 variables.

cow:
id of cow
logger:
id of logger
