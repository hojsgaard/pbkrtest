natcare(dataRep)R Documentation

Effect of antenatal care and clinic infant survival

Description

Number of babies that died in two clinics classified according top the levels of antenatal care.
(non binary) logistic regression, log linear models

Usage

data(natcare)

Format

A dataframe with 4 variables.

clinic:
The two clinics A and B
care:
The intensity of the antenatal care: 'low' and 'high'
survived:
number of babies that survived
dead:
number of babies that died

Source

Bishop, Y.M.M. (1969) Full contingency tables, logits and split contingency tables. Biometrics, 25, 119-128, Table 7.2, p. 212

References

Krzanowski, Wojtek J. (1998) An Introduction to Statistical Modelling Arnold, London,

Examples

library(dataRep)
library(lattice)
data (natcare)
