oyster(dataRep)R Documentation

Oyster data

Description

Oysters at three different locations (in the cooling water) near a power plant in Denmark. Start weight and end weight is recorded.

Usage

data(oyster)

Format

A data frame with 20 observations on the following 3 variables.

treat
a factor with levels bay drain entrance
startwgt
a numeric vector
endwgt
a numeric vector

Examples

data(oyster)
plot(endwgt~startwgt, data=oyster, col=as.numeric(treat))
