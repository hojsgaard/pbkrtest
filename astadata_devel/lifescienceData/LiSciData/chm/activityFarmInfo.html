activityFarmInfo(dataRep)R Documentation

Daily activity of cows.

Description

Information on farms on which daily activity measurements on cows were taken. A subset of the data activityCowsLong.

Usage

data(activityFarmInfo)

Format

A dataframe with 2 variables.

farm:
id of farm
owner:
owner of farm
type:
production type of farm
