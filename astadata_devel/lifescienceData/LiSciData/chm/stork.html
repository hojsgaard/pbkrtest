stork(dataRep)R Documentation

stork

Description

Number of hatching pairs of storks collected 1980-1990 for 17 European countries, geographical information and the population size and birth rate for humans.

Usage

data(stork)

Format

This data frame contains:

country:
European country.
area:
area [km2].
nstork:
Number of hatching storks pairs.
pop:
Human population [10^6].
rate:
Birth rate of human population (1000/year).

Details

Nonsense correlation

Source

Matthews, R. (2000) Storks deliver babies, (rho=0.008). Teaching Statistics, 22(2), 36-38

Examples

      
data(stork)
