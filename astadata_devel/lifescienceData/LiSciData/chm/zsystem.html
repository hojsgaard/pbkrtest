zsystem(dataRep)R Documentation

zsystem

Description

Distríbution of genotypes for blood types in the Z-system for three cattle races.

Usage

data(zsystem)

Format

This data frame contains:

race:
cattle race : rdm: danish Red [dansk=Rød dansk mælkerace], sdm: Holstein [dansk=sort brogede mælkerace], jersey
genotype:
genotype of blood-type
count:
number of cows

Details

log-linear model, two-ways table

Source

Neimann-Sørensen, A. (1958) Blood groups of cattle, Doktorafhandling, KVL, København

References

Søren Andersen (1989) Statistiske Analyse af Tælledata, Matematisk Institut, KVL p. 2, Eks.4

Examples

data(zsystem)
