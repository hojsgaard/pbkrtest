cisplatin(dataRep)R Documentation

reaction on chemotherapy with cisplatin

Description

The cisplatin data frame describe the severity of nausea for patients under chemotherapy, receiving cisplatinum or not.

Usage

data(cisplatin)

Format

This data frame contains the following columns:

cisplatinum:
no: patient received no cisplatinum, yes: received
severity:
level of serum acid phosphatase in King-Amstrong units
number
number of persons with the symptoms

Details
