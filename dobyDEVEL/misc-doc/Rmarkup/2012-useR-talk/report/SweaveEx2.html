In this example we embed parts of the examples from the kruskal.test help page into an HTML document: 

> data(airquality)

> kruskal.test(Ozone ~ Month, data = airquality)


Kruskal-Wallis rank sum test

data:  Ozone by Month 
Kruskal-Wallis chi-squared = 29.2666, df = 4, p-value =
6.901e-06


which shows that the location parameter of the Ozone distribution varies significantly from month to month.

Notice that there are 153 observations.

Finally we include a boxplot of the data : 

> boxplot(Ozone ~ Month, data = airquality)
