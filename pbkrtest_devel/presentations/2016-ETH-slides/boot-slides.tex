\frametitle{Using parametric bootstrap}
We consider the log-MLR statistic $T$ and draw $B$ parametric bootstrap samples
$t^1, \dots, t^B$ under the  estimated null model $\hat \theta_0$.


Let $f$ be the difference in the number of parameters
between the null and the general model.  

 We use this sample to estimate the parametric bootstrap p-value
  $P_{\hat\theta_0}(T \ge t_{obs})$
